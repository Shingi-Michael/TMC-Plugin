# TMC-Plugin

## Getting Started

### Dependencies
- [nvim-telescope/telescope.nvim](https://github.com/nvim-telescope/nvim-telescope)

### Installation

[lazy.nvim](https://github.com/folke/lazy.nvim):

```lua
{
  "Shingi-Michael/TMC-Plugin",
  event = "VeryLazy",
  dependencies = {
    "nvim-telescope/nvim-telescope"
  },
}

```
